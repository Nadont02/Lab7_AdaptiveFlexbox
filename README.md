# Лабораторная работа №7  
## Адаптивная вёрстка с помощью Flexbox

### Что делали
Научились делать сайты, которые красиво выглядят на любом устройстве — на компьютере, планшете и телефоне. Использовали современный инструмент CSS под названием **Flexbox** — он помогает легко расставлять блоки на странице.

### Что сделали на практике

1. **Создали папку проекта**  
   Назвали её `Lab7_Adaptiveflexbox` и положили туда все файлы.

2. **Изучили основные настройки Flexbox**  
   - Как выстраивать блоки в ряд или столбик
   - Как распределять пространство между ними
   - Как центрировать элементы без сложных расчётов
   - Как добавлять отступы между блоками

3. **Сделали три полезных примера**  
   - **Меню навигации** — ссылки, которые красиво распределяются по ширине экрана
   - **Карточки товаров** — сетка из карточек, которая сама подстраивается под размер экрана (на телефоне — одна колонка, на компьютере — несколько)
   - **Центрирование** — простой способ поставить любой блок точно по центру страницы

4. **Сделали скриншоты**  
   Сохранили картинки всех примеров в папку `img/`, чтобы было видно результат.

5. **Загрузили работу на GitHub**  
   Выложили проект в интернет, чтобы можно было показать преподавателю.
